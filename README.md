## Installation

Requires [Go 1.14+](https://golang.org/dl/)

### Linux

`$ go get -u github.com/sentinel-official/hub`

`$ cd ${GOPATH}/src/github.com/sentinel-official/hub`

`$ make install`

**Note:** To install a specific version or commit use `git checkout` command
